Explicit Keys

  • Sometimes you want to have a mapping or a sequence as a mapping key
  • Perl and many other languages don't support that natively, but you can do it in YAML
  • Sometimes you want to break your key into multiple lines for readability
--- ? simple explicit key : simple value ? inner key: inner value # this map is a key of the outer map : outer value ? this is a very long key broken into several lines : and a short value ? some explicit key # but the value is actually optional!
