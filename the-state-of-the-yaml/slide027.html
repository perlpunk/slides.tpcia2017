Flow Style

  • Sometimes YAML is too verbose
  • So you can use a style similar to JSON
--- the alphabet: - a - b - c - ... # uh, no fun

--- the alphabet: [a, b, c, d, e, f, g, h, ...]
  • No quotes necessary in many cases
  • You can have trailing commas!
